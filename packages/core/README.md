# `@danielfsousa/core`

An example of an isomorphic library from https://github.com/danielfsousa/monorepo-typescript-lib-starter

## Install

```bash
npm install @danielfsousa/core
```

## Usage

```javascript
import { add, subtract, ms } from '@danielfsousa/core'

add(1, 2) // 3
subtract(3, 2) // 1
ms('3 hours') // 10800000
```
